# Claude Telegram Bot - Документация проекта

**Дата создания:** 2026-05-03  
**Статус:** Планирование завершено ✅

## 📋 Что создано

Полная документация для разработки Telegram-бота для управления Claude CLI с минимальными требованиями.

## 🎯 Ключевые особенности проекта

- **Минимальные требования:** 2 CPU, 4GB RAM, 20GB disk
- **Простая архитектура:** Всё в Docker Compose на одном сервере
- **Без сложных систем:** Без Kubernetes, без микросервисов
- **Быстрый старт:** MVP за 3-4 месяца

## 💻 Минимальные требования

**Сервер:**
- CPU: 2 cores
- RAM: 4 GB
- Disk: 20 GB SSD
- OS: Ubuntu 20.04+

**Технологии:**
- Python 3.11+
- PostgreSQL 15 (Alpine)
- Redis 7 (Alpine)
- Docker Compose

**Стоимость сервера:** ~$10-20/месяц (VPS)

## 🛠️ Технологический стек

### Backend
- Python 3.11+
- aiogram 3.x (Telegram Bot)
- asyncpg (PostgreSQL)
- redis-py (Redis)

### Базы данных
- PostgreSQL 15 (один контейнер)
- Redis 7 (один контейнер)

### Инфраструктура
- Docker + Docker Compose
- Без Kubernetes
- Без микросервисов
- Всё на одном сервере

### Мониторинг (упрощенный)
- psutil (системные метрики)
- Docker SDK (мониторинг контейнеров)
- Простые алерты в Telegram
- Без Prometheus/Grafana

## 📦 Что НЕ включено (упрощено)

- ❌ Kubernetes
- ❌ Микросервисы
- ❌ Prometheus + Grafana
- ❌ ELK Stack
- ❌ Jaeger tracing

- ❌ Сложные CI/CD пайплайны
