## Безопасность

### Защита от атак

#### Path Traversal
```python
# Попытка выйти за пределы разрешенных директорий
/cd ../../etc
❌ Доступ запрещен

/cat /etc/passwd
❌ Доступ запрещен
```

#### Symlink Attack
```python
# Symlink на системную директорию
ln -s /etc/passwd /app/data/fake.txt
/cat fake.txt
❌ Доступ запрещен  # Разрешается в реальный путь
```

### Whitelist/Blacklist

#### Разрешенные директории
- `/app/data` - пользовательские данные
- `/app/workspace` - рабочее пространство

#### Запрещенные директории
- `/etc` - конфигурация системы
- `/sys`, `/proc` - системные файлы
- `/root` - домашняя директория root
- `/boot` - загрузчик
- `/bin`, `/sbin`, `/usr` - системные бинарники
- `/var` - переменные данные
- `/tmp` - временные файлы
