## Функционал

### Аутентификация
- Первый пользователь автоматически становится admin
- Остальные пользователи получают роль user
- Middleware проверяет аутентификацию на каждое сообщение

### Роли
- **admin** - полный доступ ко всем командам
- **user** - базовый доступ

### Команды для всех
- `/start` - регистрация
- `/help` - справка
- `/status` - статус
- `/profile` - профиль

### Команды для admin
- `/users` - список пользователей
- `/make_admin <user_id>` - назначить админа

## Как протестировать

### 1. Запустить бот
```bash
docker-compose up -d
docker-compose logs -f bot
```

### 2. В Telegram
1. Отправить `/start` - вы станете первым админом
2. Отправить `/help` - увидеть список команд
3. Отправить `/profile` - увидеть свой профиль
4. Отправить `/status` - проверить статус бота
5. Отправить `/users` - увидеть список пользователей (admin)

### 3. Добавить второго пользователя
1. Попросить кого-то отправить `/start` боту
2. Они получат роль user
3. Вы можете сделать их админом: `/make_admin <их_user_id>`
